# Personal Expense Tracker

A full-stack Personal Expense Tracker application that helps users manage their daily finances by tracking income, expenses, budgets, and spending patterns in one place.

## Features

- User Authentication (Register/Login)
- Secure Account Management
- Add Income and Expense Records
- Categorize Transactions
- Edit and Delete Transactions
- Dashboard with Financial Overview
- Monthly Expense Tracking
- Budget Management
- Transaction History
- Responsive User Interface
- Data Visualization with Charts and Graphs

## Tech Stack

### Frontend
- React.js
- React Router
- Context API
- Axios
- Tailwind CSS / CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT (JSON Web Token)
- Bcrypt.js
